telegram-detailed-notifications

Hook automation

Send detailed Telegram notifications with session information when Claude Code finishes. Includes working directory, session duration, and system info. Requires TELEGRAM_BOT_TOKEN and TELEGRAM_CHAT_ID environment variables.

How to Install

  1. Copy the config - Click the "Copy Config" button below
  2. Add to your project - Paste the code into the appropriate config file
  3. Done! - The component is now available in your AI

Configuration (click to copy):

        "matcher": "startup",
        "hooks": [
          {
            "type": "command",

          }
        ]

    "Stop": [
      {
