Quick Install

Install components directly from your terminal:

# Install all components
npx aiccli install all

# Install specific type
npx aiccli install agent <name>
npx aiccli install skill <name>
npx aiccli install command <name>
npx aiccli install mcp <name>
npx aiccli install hook <name>
npx aiccli install setting <name>

# List available components
npx aiccli list

OpenCode Integration

Use AICCLI components with OpenCode CLI:

# Create aiccli.json in your project root
{
  "components": [
    "agents/code-reviewer",
    "skills/web-scraping",
    "commands/deploy"
  ]
}

# Then run
opencode install

Or use the OpenCode component browser:

opencode browse
