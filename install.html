Install Components

Use curl to install components directly from this website

One command install! No npm, no git, no account needed.

Quick Install

Run this in your terminal:

curl -sL https://openclacode.netlify.app/cli.sh | bash -s install skills/web-scraping

Other Commands

# List all components
curl -sL https://openclacode.netlify.app/cli.sh | bash -s list

# List by type
curl -sL https://openclacode.netlify.app/cli.sh | bash -s list skills
curl -sL https://openclacode.netlify.app/cli.sh | bash -s list agents

# Search
curl -sL https://openclacode.netlify.app/cli.sh | bash -s search database
curl -sL https://openclacode.netlify.app/cli.sh | bash -s search "web scraping"

# Browse website
curl -sL https://openclacode.netlify.app/cli.sh | bash -s browse
