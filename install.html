Install Components

Browse and copy components directly from this website. No CLI required.

Private Tool: This is a private library. Components are installed by browsing this website and copying configurations manually.

How to Install

1

Browse Components

Go to the components catalog and find what you need

2

Click on Component

Open any component page to view its full configuration

3

Copy Configuration

Click the "Copy Config" button to copy the full code

4

Add to Your Project

Paste the configuration into the appropriate file in your project

Component Types

Each type has its own installation location:

Agents

Copy to your agents configuration file or agents directory.

Skills

Add to your skills directory. Skills extend AI capabilities.

Commands

Commands are triggered with "/" in your AI interface.

MCPs

Configure in your MCP settings file for external integrations.

Hooks

Add to hooks configuration for automated workflows.

Settings

Apply to your AI configuration for behavior customization.
