Install Components

Use the CLI to install components directly from this website

Quick Install

Install the CLI globally:

npm install -g aiccli

Or run directly with npx:

npx aiccli install skills/web-scraping
npx aiccli install agents/code-reviewer
npx aiccli install commands/deploy

CLI Commands

Install a component

aiccli install <type>/<name>

# Examples:
aiccli install skills/web-scraping
aiccli install agents/code-reviewer
aiccli install commands/deploy
aiccli install mcps/github
aiccli install hooks/automation
aiccli install settings/permissions

Install all of a type

aiccli install <type> all

# Examples:
aiccli install skills all
aiccli install agents all

List components

aiccli list                 # List all components
aiccli list skills          # List all skills
aiccli list agents          # List all agents

Search components

aiccli search <query>

# Examples:
aiccli search database
aiccli search web scraping
aiccli search testing

Browse website

aiccli browse

Manual Installation

1

Browse Components

Go to openclacode.netlify.app and find what you need

2

Click on Component

Open any component page to view its full configuration

3

Copy Configuration

Click the "Copy Config" button to copy the full code

4

Add to Your Project

Paste the configuration into the appropriate file

Components are installed to: .aiccli/<type>/<name>.json
