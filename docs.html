Getting Started

AICCLI is a collection of 1,700+ pre-built components for AI coding assistants. These components can be installed individually to enhance your AI's capabilities.

Quick Start: Browse components, click on any component to view details, then copy the installation command to use it in your project.

Agents

Agents are specialized AI assistants configured for specific tasks like code review, data analysis, or security auditing.

Installation

npx aiccli install agent <agent-name>

Or copy the agent configuration from any component page and add it to your agents directory.

Skills

Skills extend your AI's capabilities with specific tools and knowledge domains like web scraping, database management, or API integration.

Installation

npx aiccli install skill <skill-name>

Skills are typically added to the skills directory in your project configuration.

Commands

Slash commands provide quick access to common tasks like deployment, testing, or documentation generation.

Installation

npx aiccli install command <command-name>

Commands can be triggered with "/" followed by the command name in your AI interface.

MCPs (Model Context Protocol)

MCPs provide integrations with external tools and services like databases, APIs, and development platforms.

Installation

npx aiccli install mcp <mcp-name>

After installation, configure the MCP in your project's MCP configuration file.

Hooks

Hooks automate actions during the AI's workflow, such as running tests after code changes or sending notifications.

Installation

npx aiccli install hook <hook-name>

Hooks are added to your hooks configuration file and run automatically based on their trigger conditions.

Settings

Settings configure your AI's behavior, permissions, and environment variables.

Installation

npx aiccli install setting <setting-name>

Settings are applied to your AI configuration file and take effect on restart.

OpenCode Integration

Using with OpenCode CLI

To use AICCLI components with OpenCode:

# Install OpenCode if you haven't
npm install -g opencode

# Browse components
opencode components

# Install a component
opencode install <component-type>/<component-name>
Tip: You can install multiple components at once by creating a components.json file in your project root.
