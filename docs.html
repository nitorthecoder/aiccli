Quick Start

Install components using a single curl command:

curl -sL https://openclacode.netlify.app/cli.sh | bash -s install skills/web-scraping
No setup needed! Just copy and paste the command above. Works on any system with bash.

CLI Usage

Install a Component

curl -sL https://openclacode.netlify.app/cli.sh | bash -s install <type>/<name>

Examples:

curl -sL https://openclacode.netlify.app/cli.sh | bash -s install skills/web-scraping
curl -sL https://openclacode.netlify.app/cli.sh | bash -s install skills/javascript-pro
curl -sL https://openclacode.netlify.app/cli.sh | bash -s install skills/typescript-pro
curl -sL https://openclacode.netlify.app/cli.sh | bash -s install agents/code-reviewer
curl -sL https://openclacode.netlify.app/cli.sh | bash -s install agents/security-auditor
curl -sL https://openclacode.netlify.app/cli.sh | bash -s install commands/deploy
curl -sL https://openclacode.netlify.app/cli.sh | bash -s install commands/test
curl -sL https://openclacode.netlify.app/cli.sh | bash -s install mcps/github
curl -sL https://openclacode.netlify.app/cli.sh | bash -s install mcps/database
curl -sL https://openclacode.netlify.app/cli.sh | bash -s install hooks/automation
curl -sL https://openclacode.netlify.app/cli.sh | bash -s install settings/permissions

List Components

curl -sL https://openclacode.netlify.app/cli.sh | bash -s list
curl -sL https://openclacode.netlify.app/cli.sh | bash -s list skills
curl -sL https://openclacode.netlify.app/cli.sh | bash -s list agents

Search Components

curl -sL https://openclacode.netlify.app/cli.sh | bash -s search database
curl -sL https://openclacode.netlify.app/cli.sh | bash -s search "web scraping"
curl -sL https://openclacode.netlify.app/cli.sh | bash -s search testing

Browse Website

curl -sL https://openclacode.netlify.app/cli.sh | bash -s browse

Component Types

  • agents - Specialized AI assistants for specific tasks
  • skills - Capabilities and tools for AI
  • commands - Slash commands for quick actions
  • mcps - Model Context Protocol integrations
  • hooks - Automated workflow triggers
  • settings - Configuration and preferences

Manual Installation

Browse components at openclacode.netlify.app, click on any component, and copy the configuration manually.
