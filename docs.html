Getting Started

AICCLI is a private collection of 1,700+ pre-built components for AI coding assistants. Install components using the CLI or browse this website.

Quick Start: Install the CLI with npm install -g aiccli, then use aiccli install <type>/<name>

CLI Usage

Install a Component

aiccli install <type>/<name>

# Examples:
aiccli install skills/web-scraping
aiccli install agents/code-reviewer
aiccli install commands/deploy

List Components

aiccli list                 # All components
aiccli list skills          # Only skills
aiccli list agents          # Only agents

Search Components

aiccli search <query>

# Examples:
aiccli search database
aiccli search web scraping

Browse Website

aiccli browse

Agents

Agents are specialized AI assistants configured for specific tasks.

aiccli install agents/code-reviewer
aiccli install agents/security-auditor

Skills

Skills extend your AI's capabilities with specific tools and knowledge.

aiccli install skills/web-scraping
aiccli install skills/database-design

Commands

Slash commands provide quick access to common tasks.

aiccli install commands/deploy
aiccli install commands/test

MCPs

MCPs provide integrations with external tools and services.

aiccli install mcps/github
aiccli install mcps/database

Hooks

Hooks automate actions during the AI's workflow.

aiccli install hooks/automation
aiccli install hooks/notifications

Settings

Settings configure your AI's behavior.

aiccli install settings/permissions
aiccli install settings/model

Manual Installation

If you prefer, you can manually copy configurations from component pages:

  1. Browse components at openclacode.netlify.app
  2. Click on any component to view details
  3. Click "Copy Config" to copy the configuration
  4. Add to your project's appropriate config file
