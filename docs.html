Getting Started

AICCLI is a private collection of 1,700+ pre-built components for AI coding assistants. These components are installed manually by copying configurations from this website.

How it works: Browse components on this website, click on any component to view its full configuration, then copy and add it to your project.

Agents

Agents are specialized AI assistants configured for specific tasks like code review, data analysis, or security auditing.

Installation

  1. Browse the components catalog
  2. Click on any agent to view its configuration
  3. Copy the configuration code
  4. Add it to your agents configuration file

Skills

Skills extend your AI's capabilities with specific tools and knowledge domains like web scraping, database management, or API integration.

Commands

Slash commands provide quick access to common tasks like deployment, testing, or documentation generation.

MCPs (Model Context Protocol)

MCPs provide integrations with external tools and services like databases, APIs, and development platforms.

Hooks

Hooks automate actions during the AI's workflow, such as running tests after code changes or sending notifications.

Settings

Settings configure your AI's behavior, permissions, and environment variables.

Manual Installation

Each component page has a "Copy Config" button. Click it to copy the full configuration, then paste it into the appropriate file in your project.

Quick Fetch (Bash)

You can also fetch configurations directly using curl:

# Replace with your component path
curl -sL https://nitorthecoder.github.io/ccaitmpl/<TYPE>/<CATEGORY>/<NAME>.html | grep -oP '(?<=<code>)[^<]+(?=</code>)' | head -1
