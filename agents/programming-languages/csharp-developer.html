How to Use

  1. Install via CLI: Run npx aiccli install agents/csharp-developer in your terminal
  2. Or with OpenCode: Run opencode install agents/csharp-developer
  3. Manual install: Copy the configuration below and add it to your project's config

Installation Command:

npx aiccli install agents/csharp-developer
